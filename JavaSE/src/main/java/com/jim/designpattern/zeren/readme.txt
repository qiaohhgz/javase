对象及其责任

Student(学生)
知道自己所在的教室
知道自己下堂课的教室
从一个教室去下一个教室

Instructor(教师)
告诉学生到下堂课的教室去

Classroom(教室)
有明确地址

Direction(方向) giver(给予者)
对于给定的两个教室，指出从一个教室到另一个教室的路线

1.开始控制程序。
2.实例化室中学生的集合。
3.告诉此集合，让学生去自己下堂课的教室。
4.集合让每个学生去自己下堂课的教室。
5.每个学生都：
  a.找到自己下堂课的教室在哪里；
  b.决定怎么去;
  c.去那里。
6.完成。
